# Pixel Junior Prof for Flathub

![Pixel Junior Prof Icon](com.almaarijstudios.pxjrprof.png)

## About

**Pixel Junior Prof** is a management tool designed for teachers using the **"Pixel Informatique et Ingénierie"** textbook and the Pixel Junior student app.

With this companion application, teachers can organize and monitor the interactive activities, quizzes, and educational resources provided to their students in the Pixel Junior app, and keep track of learners’ progression in a simple classroom‑friendly environment.

- **Student tracking** – View learners’ progress and results.*
- **Activity management** – Organize and plan Pixel Junior interactive activities.
- **Educational resources** – Access extra resources to prepare and extend lessons.
- **Built for classrooms** – Features and interface designed for use by teachers.

> *Note: The exact tracking and management features available to the teacher depend on how the school and publisher deploy the solution.

The Pixel Junior and Pixel Junior Prof applications together extend the **"Pixel Informatique et Ingénierie"** textbook with a wide range of interactive activities, resources, quizzes, and mini‑games that help reinforce computing skills in a fun and progressive way.

## Key Features

- Interactive activities that favor learning by doing and discovery
- Additional educational resources to deepen what is covered in class
- Fun quizzes that help assess knowledge in an engaging way
- Educational mini‑games designed to strengthen technical skills

All activities are designed to encourage reflection, creativity, and autonomy, while remaining under the supervision of the teacher.
